新型環狀嵌段共聚物(CBC)在醫材應用的潛力
臺灣聚合化學品股份有限公司
CBC營銷及業務專員 練易鑫先生
詳情：
ViviOn™(CBC)是一種新型環狀嵌段共聚高分子(Cyclic Block Copolymer)，具有高潔淨度、低萃取/溶出量、極高的可見光和紫外光穿透度、優異的耐化性、穩定的熱氧化性、高生物兼容性，以及極低的自發熒光特性，能通過UV、EtO及伽瑪射線消毒程序，適用於押出、射出、射吹等不同的加工方式。ViviOn™(CBC)亦可藉由調整共聚物之結構，在不須添加塑化劑的情況下即能實現從軟質彈性體到硬質塑料的特性，能滿足不同的產品設計需求。綜合上述優勢，ViviOn™(CBC)可用來生產預灌封注射器、小藥瓶、高通量微孔板或UV板、醫療軟管以及藥物包材等產品。ViviOn™(CBC)通過ISO 10993生物兼容性檢驗，以及美國藥典<88> Class VI、<661>和日本藥典驗證。
